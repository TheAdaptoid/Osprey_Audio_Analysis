## Setup and Usage

### Linux

run the setup script:
```bash
bash setup.sh
```

then run the test script:
```bash
bash test.sh
```

then run the main script:
```bash
python3 osprey_audio_analysis/main.py
```

run the setup script:

.\setup.ps1
